# iOS 6 Skeuomorphic Design System

## Typography
Font stack: "Helvetica Neue", Helvetica, Arial, sans-serif
Weights: 300 (light), 400 (regular), 600 (semibold), 700 (bold)

## Colors
Primary blue: #147EFB
Primary blue dark: #0060e0
Background dark: #3d3d3d
Background medium: #4a4a4a
Surface white: #ffffff
Border gray: #c8c7cc
Text primary: #1a1a1a
Text secondary: #888888
Text on dark: #ffffff
Text muted on dark: #c8c7cc
Danger: #e84040
Success: #4cd964

## Buttons — Primary (raised, pressable)
Default: gradient #1e84fd → #0060e0, border #0050c0, radius 10px
Shadow: inset 0 1px 0 rgba(255,255,255,0.4), 0 2px 4px rgba(0,0,0,0.4)
Pressed: gradient #0050c0 → #003da0, inset 0 2px 4px rgba(0,0,0,0.5)
Disabled: gradient #a0a0a0 → #888888, no shadow

## Buttons — Secondary (subtle)
Default: gradient #f0f0f0 → #d8d8d8, border #b0b0b0, text #1a1a1a
Shadow: inset 0 1px 0 rgba(255,255,255,0.9), 0 1px 3px rgba(0,0,0,0.3)
Pressed: gradient #d0d0d0 → #b8b8b8, inset 0 2px 3px rgba(0,0,0,0.3)

## Buttons — Danger
Default: gradient #f05050 → #c03030, same shadow pattern as primary

## Toggles
Track off: gradient #d0d0d0 → #b8b8b8, border #a0a0a0
Track on: gradient #4cd964 → #34b84a, border #28a03c
Knob: white circle, shadow 0 2px 4px rgba(0,0,0,0.4), inset 0 1px 0 rgba(255,255,255,0.9)
Transition: 0.2s ease

## Inputs (recessed)
Background: linear-gradient(#d0d0d0, #e8e8e8)
Border: 1px solid #aaaaaa, radius 6px
Shadow: inset 0 2px 4px rgba(0,0,0,0.3)
Focus: border #147EFB, outer glow 0 0 0 3px rgba(20,126,251,0.3)

## Cards / Panels
Background: #ffffff
Border: 1px solid #c8c7cc, radius 10px
Shadow: 0 1px 3px rgba(0,0,0,0.3), inset 0 1px 0 rgba(255,255,255,0.8)

## Backgrounds
Linen dark: #3d3d3d + repeating-linear-gradient 45deg rgba(0,0,0,0.15)/transparent 1px
Linen medium: #4a4a4a same pattern

## Navigation sidebar
Background: linear-gradient(#4a4a4a, #2a2a2a)
Border right: 1px solid #1a1a1a
Item default: text #c8c7cc
Item hover: gradient #5a5a5a → #3a3a3a, text #ffffff
Item active: gradient #1e84fd → #0060e0, text #ffffff, inset shadow
